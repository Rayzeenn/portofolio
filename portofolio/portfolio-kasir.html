Website Admin Kasir

  • Project date: 22 Februari 2022

Deskripsi Project

Project ini adalah sebuah website admin kasir yang dirancang untuk memudahkan pengelolaan transaksi di suatu usaha. Website ini dilengkapi dengan berbagai fitur yang memungkinkan admin untuk mengelola pesanan, pelanggan, stok barang, dan barang masuk dengan efisien. Setiap fitur dirancang agar mudah digunakan dan dioptimalkan untuk mendukung operasional bisnis secara keseluruhan.

Teknologi yang digunakan

Bootstrap 5

Digunakan untuk mendesain antarmuka pengguna yang responsif dan modern.

PHP

Digunakan untuk mengelola logika aplikasi, termasuk mengakses database dan memproses data yang dikirim melalui form HTML.

MySQL

Digunakan untuk menyimpan dan mengambil data. Pada program ini, MySQL diakses melalui koneksi mysqli dalam PHP untuk menjalankan query database.

JavaScript

Digunakan untuk interaksi sisi klien, seperti penanganan modal dan integrasi tabel data dengan Simple DataTables.
