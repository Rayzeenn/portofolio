Deskripsi Projek

Projek ini adalah sebuah aplikasi catatan berbasis web yang dirancang untuk memudahkan pengguna dalam mengelola catatan mereka secara efisien. Aplikasi ini dilengkapi dengan fitur-fitur utama seperti menambah catatan baru, mengedit catatan yang sudah ada, dan menghapus catatan yang tidak diperlukan lagi.

Teknologi yang digunakan

Bootstrap 5

Bootstrap versi 5.3.1 digunakan untuk memberikan gaya dan layout yang rapi pada halaman web.

PHP

Digunakan untuk menghubungkan aplikasi dengan database MySQL, mengambil data catatan, dan menyajikannya dalam format HTML.

MySQL

Sistem manajemen basis data yang digunakan untuk menyimpan data catatan.

JavaScript

JavaScript digunakan untuk menambahkan interaktivitas pada halaman, misalnya untuk menangani event klik dan memanipulasi DOM. jQuery, sebuah library JavaScript, digunakan untuk mempermudah manipulasi ini.

CSS

Digunakan untuk menambahkan gaya khusus yang tidak disediakan oleh Bootstrap, guna menyesuaikan tampilan dan nuansa aplikasi.
