Website Digital Perpus

  • Project date: 10 Agustus 2024

Deskripsi Project

Project ini adalah sebuah website perpustakaan digital yang dirancang untuk menyimpan dan mengelola koleksi buku secara online. Buku-buku dalam perpustakaan ini diunggah oleh admin dan tersedia untuk diakses oleh pengguna. Selain itu, website ini memiliki fitur untuk menyimpan buku ke dalam daftar pribadi pengguna serta mengunduh buku sesuai kebutuhan. Dengan antarmuka yang intuitif, pengguna dapat dengan mudah menemukan dan mengakses buku-buku yang mereka cari.

Fitur

Admin

Mengelola data buku dan kategori buku

Mengekspor data data buku menjadi sebuah pdf

Menambahkan admin baru

User

Menyimpan buku

Mendownload sebuah buku

Mencari buku berdasarkan kategori yang tersedia

Teknologi yang digunakan

PHP

Sebagai bahasa pemrograman sisi server yang digunakan untuk mengelola sesi pengguna, menghubungkan ke database, dan menjalankan logika server-side seperti mengambil data dari database dan menampilkan hasil pencarian.

Bootstrap 5

Digunakan untuk mendesain halaman web yang responsif, termasuk komponen seperti navbar, modal, dan tombol.

MySQL

Digunakan untuk menghubungkan dan berinteraksi dengan database. Pada kode ini, MySQL digunakan untuk melakukan query pada tabel kategori dan buku, serta untuk mengelola data buku berdasarkan pencarian atau kategori.

JavaScript

Digunakan untuk interaktivitas di sisi klien, termasuk menginisialisasi modals dan menampilkan alert menggunakan SweetAlert.

CSS

Digunakan untuk styling seperti navbar, tombol, dan modal.
