Website Cek Jodoh

  • Project date: 1 Juli 2024

Deskripsi ProjeK

Projek ini adalah sebuah website cek jodoh yang menggunakan metode tradisional "weton" untuk menentukan kecocokan antara dua individu. Dalam budaya Jawa, weton adalah kombinasi dari hari dan pasaran kelahiran yang masing-masing memiliki nilai-nilai khusus. Website ini memungkinkan pengguna untuk memasukkan weton mereka dan pasangan untuk mengetahui tingkat kecocokan berdasarkan perhitungan tradisional tersebut.

Teknologi yang digunakan

PHP

Digunakan untuk menangani logika server-side, termasuk pengolahan data yang diterima dari form, pengambilan data dari database, dan penghitungan nilai neptu berdasarkan hari dan pasaran untuk menentukan kecocokan pasangan.

Bootstrap 5

Framework CSS yang digunakan untuk membuat antarmuka yang responsif dan konsisten secara visual. Bootstrap menyediakan gaya untuk elemen-elemen seperti form input, tombol, dan grid layout.

MySQL

Database yang digunakan untuk menyimpan data yang dibutuhkan, seperti data orang, hari, pasaran, dan nilai neptu. PHP mengakses database ini melalui koneksi MySQL untuk melakukan query dan mendapatkan data yang diperlukan.

JavaScript

JavaScript digunakan untuk memuat file bootstrap.bundle.min.js, yang mengaktifkan fitur interaktif dari Bootstrap, seperti modal dan dropdown.

CSS

Digunakan untuk memberikan gaya visual pada elemen-elemen HTML, baik melalui file CSS eksternal maupun dengan CSS inline di dalam file HTML itu sendiri. CSS digunakan untuk menata tampilan form, tombol, dan hasil output yang ditampilkan kepada pengguna.
